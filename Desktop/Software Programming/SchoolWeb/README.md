# Evergreen Academy Website

This repository contains a complete responsive static website scaffold for a modern school.

## Files

- `index.html` — Home page with hero section, features, and call to action
- `about.html` — School mission, values, and milestones
- `academics.html` — Academic programs and student support
- `admissions.html` — Admissions process, open houses, and application info
- `contact.html` — Contact details and inquiry form
- `css/style.css` — Responsive styling and layout
- `js/main.js` — Mobile navigation toggle

## Getting Started

Open `index.html` in your browser to preview the site.

For local development, use a simple static server, for example:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
